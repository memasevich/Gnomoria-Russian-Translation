# Народный Русификатор Gnomoria (Dynamic Engine)

Добро пожаловать в проект самого технологичного русификатора для легендарной стратегии **Gnomoria**. 

Этот мод был создан с нуля, чтобы решить главную проблему игры — отсутствие поддержки русских шрифтов в движке XNA. В отличие от старых методов перевода, мы не перерисовываем игровые архивы и не ломаем оригинальные файлы.

## Как это работает?

Внутри мода бьется «цифровое сердце» на C#, которое работает по принципу **Dynamic Text Rendering**:
*   **Перехват на лету:** Мод в реальном времени анализирует текст, который игра хочет вывести на экран.
*   **Живой рендер:** Если слово английское и для него есть перевод, мод блокирует стандартный вывод и мгновенно генерирует прозрачную картинку с русским текстом, используя системные шрифты Windows.
*   **Идеальная картинка:** Благодаря сглаживанию (Anti-Aliasing), текст в игре выглядит четко и читаемо при любом разрешении.
*   **Нулевая нагрузка:** Все переведенные фразы кэшируются в памяти, поэтому мод никак не влияет на производительность и FPS.

## Установка (В один клик)

1. Скачайте архив [**Gnomoria_Rus_Release.zip**](https://github.com/memasevich/Gnomoria-Russian-Translation/blob/main/Gnomoria_Rus_Release.zip) (нажмите на ссылку, а затем на кнопку "Download raw").
2. Распакуйте содержимое архива в корневую папку игры (там, где находится `Gnomoria.exe`).
3. Запустите файл **`Patcher.exe`** из папки с игрой.
4. Дождитесь надписи `Injection successful!` и закройте окно.
5. Запускайте Gnomoria через Steam. Игра теперь полностью на русском!
