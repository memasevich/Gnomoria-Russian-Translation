# Народный Русификатор Gnomoria

Мод-русификатор, который переводит интерфейс, профессии, предметы и подсказки в игре **Gnomoria** на русский язык.

## Установка (В один клик)

1. Скачайте архив **`Gnomoria_Rus_Release.zip`** из раздела [Releases](../../releases).
2. Распакуйте **содержимое** архива в главную папку игры (туда, где находится файл `Gnomoria.exe`).
   * *Обычно путь в Steam выглядит так: `C:\Program Files (x86)\Steam\steamapps\common\Gnomoria`*
3. Запустите файл **`Patcher.exe`** из папки с игрой.
4. Откроется черное окно. Дождитесь появления надписи `Injection successful!`. 
   * *(Мод автоматически создаст резервную копию оригинального файла `Gnomoria.exe.backup`)*.
5. Закройте черное окно, запускайте игру через Steam и наслаждайтесь!

## Как помочь улучшить перевод?

В игре очень много текста, и некоторые редкие фразы (или новые события) могут все еще отображаться на английском. Вы можете перевести их сами прямо во время игры!

Мод работает по системе "живого перевода". Если он видит незнакомое английское слово, он записывает его в файл словаря.

**Инструкция по переводу:**
1. Если вы увидели в игре английский текст, закройте игру.
2. В папке с игрой откройте файл **`Gnomoria_en_ru.json`** с помощью обычного Блокнота.
3. В самом низу списка вы найдете непереведенные строки. Они выглядят так:
   `"dirt block": "dirt block"`
4. Просто замените английский текст **справа** (после двоеточия) на русский:
   `"dirt block": "земляной блок"`
5. Обязательно сохраните кавычки `""` и запятую `,` в конце строки!
6. Сохраните файл и запустите игру. Текст мгновенно обновится на русский.

Если вы перевели много новых строк, вы можете скинуть свой файл `Gnomoria_en_ru.json` в сообщество Steam или создать Issue в этом репозитории, чтобы помочь другим игрокам!
