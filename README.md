# Gnomoria Russian Translation (Dynamic Engine)

Русификатор для Gnomoria, работающий на базе динамического рендеринга текста. 

В отличие от классических методов, этот мод не требует декомпиляции игровых архивов или перерисовки `.xnb` шрифтов. Он перехватывает вызовы отрисовки и рендерит кириллицу на лету через системные шрифты Windows.

## Как это реализовано
- **Hook Engine:** Используется Harmony для перехвата `SpriteBatch.DrawString` (поддерживаются `string` и `StringBuilder`).
- **Dynamic Rendering:** Переведенный текст отрисовывается в прозрачный `Bitmap` (GDI+) и конвертируется в `Texture2D`.
- **Performance:** Текстуры кэшируются. Генерация происходит один раз для каждой уникальной строки.
- **Auto-collect:** Непереведенные строки автоматически дампится в JSON-словарь во время игры.

## Установка
1. Скачать [**Gnomoria_Rus_Release.zip**](https://github.com/memasevich/Gnomoria-Russian-Translation/blob/main/Gnomoria_Rus_Release.zip).
2. Распаковать в корень игры (там где `Gnomoria.exe`).
3. Запустить `Patcher.exe` для инъекции хука.
4. Готово. Оригинал будет сохранен как `Gnomoria.exe.backup`.
